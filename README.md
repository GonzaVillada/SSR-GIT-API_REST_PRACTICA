# SSR-GIT-API_REST_PRACTICA
## Éste repositorio fue creado para realizar en grupo el trabajo de repaso de la materia Practica Profesionalizante III.

### Pasos para ejecutar la API REST y SSR.

- Descargar e instalar Visual Studio Code (VSC).
- Entrar a nuestro GitHub: https://github.com/GonzaVillada/SSR-GIT-API_REST_PRACTICA
- Crear una carpeta.
- Abrir CMD en esa carpeta.
- Clonar el repositorio con: git clone https://github.com/GonzaVillada/SSR-GIT-API_REST_PRACTICA

**Para la API REST**
- Descargar e instalar XAMPP.
- Ejecutar Apache y MySQL dentro de XAMPP.
- Importar la base de datos en PHPMyAdmin.
- Abrir la carpeta API REST en VSC.
- Dentro de VSC, abrir una nueva terminal y colocar: php -S localhost:8000

**Para el SSR**
- Descargar e instalar node.js
- Abrir la carpeta SSR en VSC.
- Dentro de VSC, abrir una nueva terminar y colocar: node server.js

## Visualizar la página
Colocar en el buscador: localhost:3000
